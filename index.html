Summary: Latitude vs. X

This project's aim is to compare different weather elements and see if there is a correlation with the city's distance from the equator. The four elements being compared are temperature, humidity, cloudiness, and wind speed. The data was sourced from the Open Weather API and graphed using Matplotlib.

The navigation bar above links to the plot for each element, a comparison page for all four, and the raw data set. The visualizations on the right side of this page also links to a closer look at each element.
